index.php
	-> Página principal de la aplicación.
	-> Al costado derecho se presentan dos opciones, para traducir de partituras a audios o viceversa.
	-> Al costado izquierdo se presenta una imagen, donde pudiese ser un logo del proyecto.

par2tab.php
	-> Se presenta una imagen, la cual representa lo que debe mostrar la cámara del dispositivo movil.
	-> Posee un botón "ESCANEAR", para comenzar este proceso.
	-> Ofrece botón para descargar audio una vez realizado el escaneo.
	-> Opción para volver a Menú Principal.

tab2par.php
	-> Ofrece botón "CARGAR AUDIO" para realizar dicha acción.
	-> "CONVERTIR" se utiliza para conocer la partitura del audio, una vez haya sido cargado.
	-> Se presenta una imagen que representa la partitura que se ha detectado del audio.
	-> Se entrega la opción de descargar la partitura mediante botón "DESCARGAR".
	-> Opción para volver a Menú Principal.
